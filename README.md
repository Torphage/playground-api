# playground-api

`playground-api` is a Rust API project where I challenge myself by trying new technologies, ideas, and development approaches.

The name *playground* comes from the fact that this project is meant to be a place for experimentation and learning.

The project is guided by a few principles:

- Clean architecture
- Domain-driven design
- Test-driven development
- Clean code

The aim is to build an API that is easy to extend and maintain, while still being performant and secure.

## Goals

The goal is not necessarily to build a complete product, but to use the project as a way to experiment, learn, and improve.
